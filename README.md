# Webpack
### Commonly used techniques from three aspects:
- improving the building speed
  - thread-loader
  - cache-loader
  - Hot update
  - exclude & include

- reducing the size of the packaged files
  - Minify CSS Code
  - Minify Javascript Code
  - tree-shaking
  - source-map
  - Build Analyzer

- improving the user experience
  - Module lazy loading
  - Gzip
  - base64
  - Properly configure hash
